Описание порядка работы:
1. Для начала работы с прогой нужно взять названия позиций, цены и описания позиций и положить в файлы Positions.txt, Prices.txt и Description.txt соответственно. Строки одного файла должны соответствовать строкам другого и третьего. Допускаются пустые строки для удобства проверки, прога это понимает.
Сделать это нужно 1 раз (уже сделано). По идее эти файлы можно прибить после первого запуска, но если понадобится что-то поменять, придётся их полностью по новой делать.
2.Запускаем прогу, она автоматом делает файл базы db.json, куда лезть не стоит и файл с индексами позиций Index.csv. Этот Index.csv открываем экселем, но не напрямую, а через вкладку "Данные" - "Из текстового файла/csv". Там выбрать кодировку UTF-8. Если открыть напрямую- получишь крякозябры. Чтобы не париться, импортированные данные сохраняем как родной файл экселя и радуемся жизни.
3. Прога начинает запрашивать данные - индекс позиции и количество. Вводим, когда надоест в номер позиции вводим "0", прога сохраняет файл out.txt и закрывается.
4. Открываем файл out.txt в Ворде и радуемся жизни.
5. Если надо что-то поменять в описаниях/ценах/позициях, лезем в соответствующие файлы, меняем. Потом удаляем файл db.json и файл index.csv. При следующем запуске прога поймёт, что базу надо обновить и пересоздаст базу из новых файлов.
Соответственно повторяем п.п.2 для получения нового индекса в экселе.

P.S. По-хорошему, сюда можно прикрутить и обновление данных из программы, и базу данных, например, postgres, чтоб в файле не копаться, и морду красивую, но это за отдельные $$$.
